# 筋トレタイマー & トレーニングメモアプリ 

##  概要
筋トレ中に使える、カウントダウンタイマーと
重量・回数・セット数を記録できるアプリです。

- トレーニングのインターバル管理
- 種目ごとの記録
- 編集 / 削除機能つき
